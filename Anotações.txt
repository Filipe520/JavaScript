                                Aula 10 - EVENTOS COM MAUSE - DOM

                                     EVENTO - mouseenter

      evento é quando colocar o mause em cima de uma elemento( exemplo - um bloco article ) 
    quando passar o mause em cima dele vai disparar um alerta falando que o mause tá em cima
         mas esse alerta só vai aparecer ser agente falar para mostrar no console.

                                     EVENTO -maousemove

    Vai disparar um alerta falando que o mause tá ser movendo dentro do elemento que tá selecionado.

                                     EVENTO - mousedown

    Quando clicar no elemento vai disparar um alerta é quando solta o mause vai ser outro EVENTO.

                                    EVENTO - mauseup

    Quando solta o clique vai dispara esse evento

                                    EVENTO - click

    Vai ser quando clica rápido.

                                    Evento - mouseout

    Vai ser quando tirar o mause fora do elemento ai vai dispara esse alerta.

<body>
    <h1>Tabuada de Multiplicação</h1>
    <input type="number" name="tabuada01" id="tabuada01">
    <input type="number" name="tabuada02" id="tabuada02">
    <input type="button" value="Somar" onclick="somar()">
    <p id="res">Resultado</p>

    <script>
        function somar() {
            let st1 = window.document.getElementById('tabuada01')
            let st2 = window.document.getElementById('tabuada02')
            let res = window.document.getElementById('res')
    
            let n1 = Number(st1.value)
            let n2 = Number(st2.value)
            
            res.innerHTML = `A soma entre ${n1} e ${n2} é igual a ${n1*n2}`
        }
        
    </script>
</body>
